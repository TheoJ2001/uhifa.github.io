adds pins to control and read the status of the cylinder and grabber

Precondition
is called in setup, needs to be used before Grabber::config() and type has to be VACUUM or CLAW
Parameters
typedetermines what type of grabber is used
rtd_pinpin for sensor that becomes HIGH when a Grabber object is retracted
ext_pinpin for sensor that becomes HIGH when a Grabber object is extended
push_pinpin for actuator that opens valve
hold_pinpin for sensor that turns HIGH when a Grabber object is holding something
grab_pinpin for actuator that engages grabber

◆ get()

int16_t Shuttle::get ( int8_t  mode)
virtual

gets the state of a sensor or actuator

Precondition
mode must be POSITON, MOVING, DELIVERING, SAFE, SEQUENCE_INDEX to get a return value other than 0
Parameters
mode
Returns
int16_t

POSITON returns the last stop the arm passed, MOVING returns true if arm is moving, DELIVERING returns true if the arm is delivering something, SAFE returns true if arm.get(SAFE) is true, SEQUENCE_INDEX returns the current step of the delivery sequence of endDeliv and beginDeliv
